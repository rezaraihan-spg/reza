# Slot Demo

Sebuah demo mesin slot frontend-only (tanpa uang nyata). Buka `index.html` di browser untuk menjalankan.

Cara pakai:

1. Buka file `c:\Users\user\Documents\Web AI\slot_demo\index.html` di browser.
2. Tekan `SPIN` untuk memutar. Gunakan tombol `Bet` untuk mengubah taruhan.

Catatan: Hanya simulasi edukasi/hiburan.
